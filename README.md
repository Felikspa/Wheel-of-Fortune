# Wheel of Fortune (Flutter)

> 本 README 已按当前代码重写（更新于 2026-04-06），覆盖上次 README 之后新增的全部核心能力。

一个移动端随机决策应用，已从“单转盘抽取”升级为“多模式抽取引擎 + 多转盘管理 + DSL 编解码 + 全局背景图系统”。

## 1. 上次 README 之后新增的核心特性

- 抽取模式从单一转盘扩展为 4 模式：`wheel` / `coin` / `dice` / `card`。
- 抽取模式按转盘独立记忆：每个转盘可保存自己当前显示模式。
- 结果按“转盘 + 模式”隔离记录：不同模式互不覆盖。
- 转盘手势旋转升级：左右边缘上下滑可加速/减速，旋转中长按边缘触发刹车。
- 转盘交互升级：支持缩放/平移、视口状态持久化（旋转角、缩放、位移、光影参数）。
- 新增高速不稳定模拟（视觉抖动 + 触感节奏变化）和实时转速 HUD（RPM）。
- 新增全局背景图系统：选图、换图、清除、启用开关、透明度、模糊度。
- 设置背景图时，背景光源层自动隐藏（背景图不透明度为 1 时可完全覆盖底层视觉元素）。
- 调色板扩展：新增 `pink`、`transparent`，并强化液态玻璃风格。
- 抽屉升级：在抽屉内直接切换“转盘 + 抽取模式”。
- 新增硬币模式双面配对记忆（可自动补全历史搭档）。
- 新增骰子模式面映射系统（`D6/D8/D12/D20`），支持映射继承与校验。
- 新增卡牌模式：洗牌、翻牌、`Reveal All` 开关。
- 快捷导入增强：支持中英文标点归一、键值字段、动态 schema、自定义字段。
- 管理页底部新增小型液态玻璃签名组件：`ljx&hly`（居中、半屏宽、最大拉伸感）。

## 2. 功能总览

### 2.1 四种抽取模式

1. `Wheel`（转盘模式）
- 指针固定朝上，目标落点由抽取引擎计算。
- 支持点击 `Spin` 目标旋转。
- 支持边缘手势触发自由旋转与旋转中二次加/减速。

2. `Coin`（硬币模式）
- 从当前转盘项目里选择 A/B 两面。
- 只选单面时会尝试从历史搭档自动补齐另一面。
- 不允许 A/B 选择同一项目。

3. `Dice`（骰子模式）
- 支持 `D6/D8/D12/D20`。
- 每个面映射到一个项目，禁止重复映射。
- 映射不完整或重复时禁止掷骰。

4. `Card`（卡牌模式）
- 洗牌后才可翻牌。
- 可配置 `Reveal All`：翻开一张后展示单张或全部正面。

### 2.2 概率模式

- `equal`：等概率。
- `weighted`：按权重抽取（空/非正权重按 1 处理）。
- `softAntiRepeat`：基于最近中奖窗口做软抑制，降低连续重复概率。

说明：
- `wheel`/`dice`/`card` 使用统一抽取引擎（均可受概率模式影响）。
- `coin` 为二选一随机抛掷逻辑。

### 2.3 多转盘与管理

- 新增/删除/重命名转盘。
- 转盘项目增删改查。
- 项目拖拽排序（`ReorderableListView`）。
- 项目字段：`title`、`subtitle`、`tags`、`note`、`colorHex`、`weight`。
- 单转盘最大 100 项。

### 2.4 抽屉与页面结构

- 主页面仍为双页：抽取页 + Manage 页（左右滑动切页）。
- 抽屉内可直接：
  - 选择当前转盘；
  - 切换当前转盘显示模式（Wheel/Coin/Dice/Card）。
- 忙碌状态（旋转或辅助动画）时会锁定部分交互，避免冲突。

## 3. 转盘模式交互细节（重点）

- 边缘触发区域：圆盘左右侧边缘带。
- 左右侧上下滑映射为统一切向速度投影（两侧方向自动修正）。
- 支持两类旋转：
  - 目标旋转（点击 `Spin`，按目标中奖项落点结束）。
  - 自由旋转（手势触发，按摩擦衰减停止）。
- 旋转中可边缘手势二次拨动，动态调速。
- 旋转中长按边缘可刹车（加大摩擦/加速收敛）。
- 高速区间触发不稳定效果：
  - 视觉：轻微抖动、位移扰动；
  - 触觉：脉冲节奏随速度提升。
- 视口状态持久化：切换转盘后可恢复缩放、位移、角度与光影参数。

## 4. 全局背景图系统

背景图设置入口在 Manage 页的当前转盘设置卡片中，但配置作用域为全局（写入 `AppSettings`）：

- 选择图片（系统文件选择器）。
- 更换图片（旧图文件会清理）。
- 清除图片。
- 启用/禁用背景图开关。
- 透明度调节（`0.0 ~ 1.0`）。
- 模糊度调节（`0 ~ 24` sigma）。

渲染规则：
- 当背景图启用且路径有效时，背景图覆盖全屏背景层。
- 当背景图启用时，原背景光源层自动隐藏。

## 5. 快捷导入与 DSL 编解码

### 5.1 完整转盘导入/导出

- 导出：当前转盘可导出为 DSL 文本并复制到剪贴板。
- 导入：支持新建转盘导入（不覆盖原转盘）。
- 支持格式：
  - `csv`（`,` 字段分隔，`;` 项分隔）
  - `pipe`（`|` 字段分隔，按行分项）
- 头部元信息：
  - `@format` `@name` `@mode` `@spinDurationMs` `@palette`

### 5.2 Quick Import（追加到当前转盘）

- 文本直接解析为项目并追加，不新建转盘。
- 支持中英文标点自动归一（例如 `，；：`）。
- 支持键值字段：如 `site:楼下,color:blue`。
- 首条记录可建立字段 schema，后续同位置字段自动映射。
- 非核心字段会进入 `customFields`。
- 部分失败策略：有效项继续导入，错误按行反馈。

## 6. 本地化与主题

- 语言：系统 / 中文 / 英文。
- 系统语言非 `zh/en` 时回退英文。
- 主题：系统 / 浅色 / 深色。

## 7. 持久化与数据结构

- 本地数据库：Isar。
- 主要集合：
  - `WheelRecord`
  - `WheelItemRecord`
  - `AppSettingsRecord`
- `AppSettings` 中持久化内容包含：
  - 语言覆盖、主题；
  - 每转盘显示模式；
  - Coin/Dice/Card 的每转盘配置；
  - 全局背景图配置（路径/开关/透明度/模糊）。

## 8. 技术栈

- Flutter 3.41.x / Dart 3.11.x
- 状态管理：Provider + ChangeNotifier
- 存储：Isar
- 本地化：`gen-l10n`
- 文件选择：`file_picker`
- 路径与文档目录：`path_provider`
- UI：Material 3 + `liquid_glass_widgets`
